This folder contains all the scripts for topic modelling of the tweets.

1) Get tweets from twitter analysis dashboard in .csv format.
2) First run Python Script on original tweets dataset to extract features and basic preprocessing -- TweetTextPreProcessing.py
2) Run R script to perform text pre processing. -- TweetsPostPreProcess.R
3) Run Python Script for building dictionary as well as spell corretction of words in tweets. -- Build Dictionary.py and SpellcheckAndCorrection.py
4) Run R script -- Topic Modelling.R to get topic modelled tweets dataset and other related files.
5) Run TweetsTMAnalysis to get insights into the topics
